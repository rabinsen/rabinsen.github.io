  2. What is the difference between a method and function?

    Ans: Function is a block of code written to perform specific task defined using function keyword followed by function name and can live by its own. Methods is a function associated with an object property. The method operates on the data contained in a class. The function can be called directly
    by its name whereas method can be called by object name and method name using dot notation.

  3. What does 'this' refer to when used in a Java method?

    Ans: 'this' refers to the object of current class.

  4. What does 'this' refer to when used in a JavaScript method?

    Ans: 'this' refers to the owner object of that method.

  5. What does 'this' refer to when used in a JavaScript constructor function?

    Ans: In constructor function 'this' refers to the new object when a new object is created.

  6. Assume object x is the prototype for object y in Javascript. Object x has a method f( ) containing keyword 'this'. When f is called by x.f( ), what does 'this' refer to?

    Ans: 'this' refers to x.

  7. What is a free variable in JavaScript?

    Ans: Free variable in JavaScript are the variables in the function that are not the parameters or declared in that particualar function but rather comes from surrounding function.

  10. Write a Javascript function that takes any number of input arguments and returns the product of the arguments.

    Ans: function findProduct(){
    var product = 1;
    for(let i=0; i < arguments.length; i++){
    product *= arguments[i];
    }
    return product;
    }
    var result = findProduct(1, 2, 3, 4, 5, 6, 7);
    document.getElementById("result").innerHTML = result;
