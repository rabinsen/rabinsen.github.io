4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Ans: Unobtrusive Javascript is the way of writing Javascript code in which we properly separate Document Content and Script Content
and thus allowing us to make a clear distinction between them. The reasons for using it are: it separates the website
into 3 major categories: content(HTML), presentation(CSS) and behavior(JavaScript) and make code clean. Also it allows us to work on JavaScript
without touching either HTML or CSS. Thus, it makes out code less error prone, easy to update and to debug.
