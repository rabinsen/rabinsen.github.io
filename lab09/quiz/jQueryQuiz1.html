jQuery Quiz 1

1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?
Ans: When the jQuery function is invoked with a CSS selector, it will return a jQuery object wrapping any element that match this selector. The jQuery object does not represen the DOM method.
2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red. 
Sample HTML:
            <body>
            <h1>abc</h1><br><div>
             <h1>div-1</h1>
             <h1>div-2</h1>
             </div>
             <h1>xyz</h1>
            </body>
Ans: $("div h1").css("background-color", "red");
