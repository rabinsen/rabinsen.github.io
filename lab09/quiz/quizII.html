1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent. 

                //code fragment 1
                $("li").each(function(idx, e) {
                    $(e).css(“color”, “yellow”); });
                //code fragment 2
                $("li").each(function() {
                    $(this).css(“color”, “yellow”); });
                //code fragment 3
                $("li").each(function(idx) {
                    $(this).css(“color”, “yellow”); });
  1. Test this sentence 1
  2. Test this sentence 2
  3. Test this sentence 3
  4. Test this sentence 4

Ans: They all are the same. They can have different parameters and still be equivalent because in the first fragment e in the function refers to value which points to the current element on which the event is created. Similarly on second and third code fragments 'this' refers to the same current element on which the event is created. This is why all three code fragments are same.
